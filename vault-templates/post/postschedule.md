## How to Use

- Agent adds draft entries at 08:00 daily (status: `draft`)
- Owner reviews and approves via Lark (status becomes `approved`)
- Agent publishes at scheduled time if status is `approved`
- Never publish anything with status `draft` or `pending-approval`

## Status Values

| Status | Meaning |
|---|---|
| `draft` | Created by agent, not yet reviewed |
| `pending-approval` | Tier 1 or Tier 2 escalation sent, awaiting owner |
| `approved` | Owner approved, ready to publish at scheduled time |
| `published` | Live on platform вЂ” do not modify |
| `rejected` | Owner rejected вЂ” move to archive |
| `held` | Deferred to new date/time (see Notes column) |
