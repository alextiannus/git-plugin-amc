# TikTok SEO — F&B Playbook

TikTok is increasingly a search engine, especially for food content. Gen Z uses TikTok search the way Millennials use Google. Optimising for TikTok SEO means more discoverability long after a video is posted — not just the first 48 hours of the algorithm push.

---

## When to Use

Apply when writing TikTok video titles, captions, and in-video text overlays. Use alongside the core `tiktok` platform skill.

---

## Where TikTok Reads Keywords

TikTok's algorithm extracts keywords from 5 places, in order of weight:

1. **Video title** (typed in the "Describe your video" field) — highest weight
2. **Spoken words** (auto-transcribed by TikTok ASR) — very high weight
3. **On-screen text overlays** — high weight
4. **Caption / description** — medium weight
5. **Hashtags** — lower weight than most creators think

**Implication:** Say the keywords aloud in the video. TikTok hears it.

---

## Keyword Strategy for F&B

### High-Intent Food Search Terms
Target keywords people actually search on TikTok:

```
"best [cuisine] in [city]"
"[dish name] recipe"
"[dish name] near me"
"authentic [cuisine] restaurant"
"[city] food tour"
"Chinese food [city]"
"is [dish] worth it"
"[dish] review"
```

### How to Use Keywords

**Title formula:**
```
[Primary keyword] + [Secondary keyword] + [Hook or qualifier]
```
Example: `"Authentic char siu in Toronto — is it worth the hype?"`

**Spoken script opening (first 3 seconds):**
Say the primary keyword out loud immediately.
`"If you're looking for the BEST dumplings in [city]..."`

**Caption formula:**
```
[2–3 sentences expanding on the title] + [primary keyword used naturally] + [3–5 hashtags]
```

**On-screen text:**
Repeat the primary keyword as a text overlay in the first 3 seconds. TikTok's OCR reads it.

---

## Hashtag Rules for F&B TikTok

- Use **3–5 hashtags** — same as Instagram, more is not better
- Priority order: `#[specific dish]` > `#[cuisine]food` > `#[city]food` > `#foodtok`
- **Always include `#foodtok`** — it's the largest F&B community hashtag and actively recommended by TikTok
- Avoid: `#fyp`, `#foryoupage`, `#viral` — no evidence these affect ranking

```
Good: #charsiu #chinesefood #torontofood #foodtok
Bad:  #fyp #viral #food #yummy #delicious #foodie #foodlover
```

## Audio Selection

| Audio type | SEO impact | Reach impact |
|---|---|---|
| Trending audio (from Trending tab) | None | High (+2× distribution) |
| Original audio (no music) | High (TikTok transcribes spoken words) | Medium |
| Popular but not trending | Low | Low |

**Best practice for F&B:** Use trending audio for reach-focused content, original audio (talking head or voiceover) for SEO-focused content. Combine by using trending audio as background and speaking keywords clearly over it.

## Content Formats with Highest F&B Search Ranking

| Format | Search performance | Why |
|---|---|---|
| "Best [dish] in [city]" review | ⭐⭐⭐⭐⭐ | Exact-match search queries |
| Recipe walkthrough | ⭐⭐⭐⭐⭐ | High save rate = sustained push |
| Restaurant tour / order walkthrough | ⭐⭐⭐⭐ | "What to order at [restaurant]" queries |
| Behind the scenes kitchen | ⭐⭐⭐ | Authenticity signal, saves |
| Pure entertainment / trend | ⭐⭐ | Short-term reach, low search value |

## Notes

- TikTok's search algorithm updates frequently — test monthly
- Videos rank in search for 90+ days if they have sustained watch time and saves
- For ZH-language content targeting Chinese diaspora, use ZH keywords AND EN keywords in caption — TikTok serves both
- Never keyword-stuff the caption; TikTok's spam filter deprioritises it
